// expertise

Hello! I'm Ekaterina, a passionate web developer with a deep love for creating elegant and functional websites.
My expertise lies in designing and crafting responsive and visually appealing websites using Figma for high fidelity prototypes and the latest web technologies such as Bootstrap, Tailwind, MUI and React. In my projects I also use Node.js and Express.js to build an efficient backend infrastructure. I have experience working with MongoDB, a NoSQL database.
Feel free to explore my portfolio and get in touch to discuss your next project!

{ HOLIDAZE }

A holidays accomodation website allowing a customer to book listed venues and a venue owner to manage properties. The website features minimalistic design and colour palette that encourages adventurous mindset and desire to travel. The app is built with MUI.

{ E-commerce website }

An e-commerce website selling everything from perfume to cars. The app allows a user browse through the items, get more details about them, add them to the cart and checkout in the end. A customer has an opportunity to contact the store by writing a message. The app is built with MUI.

{ Auction House }

An online auction platform that allows users to both showcase their items for sale and bid on others'. Upon registration, each client receives 1000 points to use on bidding and gets points for every item sold. Auction House offers a user-friendly interface where participants can easily navigate through the items, place their bids, and monitor the bids of others. The web application is built again with Express.js on top on Node.js.

{ Rainy Days }

An e-commerce website selling outdoors jackets of various kinds. The target group includes middle-aged people ready for new challenges and fascinating adventures. The choice of breath-taking outdoors images, natural colours and clean geometric fonts - all that designed to attract a customer.

{ Community Science Museum }

A website for a local science museum which has various exhibition spaces which it strives to make as much fun as possible. The website is directed at both children interested in science and families looking for a place to spend their time together while learning something new.

{ Oslo Sup }

A blog of a paddle board enthusiast living currently in Oslo. The website invites you to jump on the board and explore the ocean with its strong marine vibe and inspiring ocean images. The posts cover a wide range of topics dedicated to the activity.
